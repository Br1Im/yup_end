# Yup End

Final version of Yup platform with updated landing page and improved interface.

## Tech Stack

- **React**
- **Next.js**
- **TypeScript**
- **Tailwind CSS**

## Features

- Updated landing page
- Improved UI/UX
- Responsive design

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- npm or yarn

### Installation

```bash
git clone https://github.com/Br1Im/yup_end.git
cd yup_end
npm install
```

### Running

```bash
npm run dev
```
